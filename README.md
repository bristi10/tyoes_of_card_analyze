# tyoes_of_card_analyze
This project analyzes and compares the usage, features, and transaction patterns of credit and debit cards across payment networks like Visa.

## Tools used
-Excel
-Power BI
-Pandas
-Jupyter Notebook

## Dataset
 It shows customer card information, card types, security features, and basic risk indicators.

 ## Analytical Question
1. Most popular card brand
2. Average credit limit by brand
3. Chip vs non-chip card analysis
4. Dark web risk analysis
5. Clients with most cards issued
6. Credit limit distribution
7. PIN Change Analysis
